# 서울 은행 서바이벌 🍂

가을철 서울 도심에서 하늘에서 떨어지는 은행열매들을 피해 최대한 오래 버티는 HTML5 아케이드 게임입니다.

## 🎮 게임 플레이

- **목표**: 떨어지는 은행열매를 피해 최대한 오래 버티기
- **조작**: 방향키 또는 WASD로 이동, 스페이스바로 점프
- **특징**: 다양한 타입의 은행열매와 파워업 시스템

## 🌟 게임 특징

### 은행열매 타입
- **일반 은행**: 기본 데미지
- **썩은 은행**: 높은 데미지 + 냄새 디버프
- **거대 은행**: 큰 데미지, 느린 속도
- **연쇄 은행**: 폭발 시 작은 은행들이 사방으로 튀어나감

### 파워업 아이템
- **마스크 (M)**: 냄새 데미지 감소
- **우산 (U)**: 은행 한 번 완전 방어
- **부츠 (B)**: 이동속도 증가
- **방독면 (G)**: 썩은 은행 완전 무시
